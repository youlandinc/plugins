## What this example demonstrates

- Connector check passes for a non-admin user; role is noted but does not block early steps.
- PAUSE gate: no write tools called until Step 6 confirm.
- Empty-category guard fires (18% fill rate < 20%) and asks before synthesizing — user picks "continue anyway".
- Idempotency check uses `all_monday_api` GraphQL + client-side description match, not `search`.
- `create_dashboard` fails on permission error → non-admin fallback prompt shown.
- User picks option (a) → `create_doc` succeeds; doc carries skill-id comment for future idempotency.
- Summary line references "doc" instead of "dashboard" so the user knows which artifact type was created.
- Safety rail held: no stage edits, no amount writes, no deletes.
