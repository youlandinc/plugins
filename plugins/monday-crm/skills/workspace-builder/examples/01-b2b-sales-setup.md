## What this example demonstrates

- Open-ended description → archetype match without exposing internal taxonomy.
- Single follow-up question skipped (enough signal in the initial description).
- Default mode: every `create_*` confirmed via the proposal step.
- Idempotency: searched for existing "Deals" board before creating.
- Setup summary doc published with `Generated by Claude` footer + `<!-- claude-skill-id: workspace-builder -->`.
- Cross-skill handoffs: daily-briefing + data-cleanup suggested as next steps.
- Safety rail held: no deletes, no overwrites, no amount-column writes, single workspace scope.
