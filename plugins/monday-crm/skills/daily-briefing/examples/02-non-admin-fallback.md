## What this example demonstrates

- **Multiple CRM boards** → `AskUserQuestion` with "all of them" option.
- **Non-English column names** (French "Gagné"/"Perdu") → resolved by `is_done: true`, not by text.
- **>200 active deals** → `board_insights` fallback + sampling note + per-deal detail only on actionable buckets.
- **Permission fallback:** `create_update` on admin-owned item fails → degrade to `create_doc` in personal workspace. No panic, no stop.
- **Proactive scoping:** IC can only post on deals they own. Others' deals are **listed** but not **written to** — safety rail + permission model aligned.
- **Sampling flag** appears in the brief body so the user knows it's not the full board.
- **Under 20 proactive writes** → individual confirms (well, session-level approve-all, no per-deal re-confirm).
