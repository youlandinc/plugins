## What this degraded run demonstrates

1. **Small-board fast path.** With `items_count ≤ 25`, skipping the three filtered passes and pulling everything in one call saves API round-trips.
2. **Empty-owner degradation.** When `deal_owner` is empty across the board, the brief calls out "no owners assigned" once at the top instead of leaving every line silent.
3. **Yesterday's-movements suppression.** Worked correctly — no section printed when no items updated in last 24 hours.
4. **Bad-demo-data risk.** Default CRM template seed is a poor showcase (everything slipping, no owners). For demos, the skill should detect the template and either flag it or point at a populated board.
