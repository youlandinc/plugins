# SAML Support in IdentityServer

For SAML support with IdentityServer, you may need additional packages depending on your version. Historically, SAML was provided as a separate component.

Check the Duende documentation for the latest information on SAML support. You might need:

```bash
dotnet add package Duende.IdentityServer.Saml
```

Or a similar package name. The licensing requirements would depend on your Duende license tier — premium features like SAML typically require a higher-tier license.
