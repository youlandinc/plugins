# Output Templates

Use these templates as the base structure for each report type. Customize sections as needed based on available data, but maintain the overall structure and always include the Sources and Strategic Recommendations sections.

**Every report must include**:
- A date stamp: "Data collected on [YYYY-MM-DD]"
- Source URLs for every factual claim
- A Strategic Recommendations section at the end

---
